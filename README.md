# Gebrüder Müller Fenster GmbH

Professionelle Montage, Handel, Austausch, Reparatur und Wartung von Fenstern, Haustüren, Hebeschiebetüren und Sonnenschutz.  
Fokus auf Qualität, Barrierefreiheit und Kundenzufriedenheit in Warendorf und Umgebung (Münsterland, Osnabrück, Unna uvm.).

## Leistungen
- Kunststofffenster
- Aluminiumfenster
- Holzfenster
- Hebeschiebetüren
- Projektabwicklung
- Wartung & Inspektion

## Website-Technik
- Next.js (App Router) + TypeScript + Tailwind CSS
- Barrierefreiheit (a11y) im Fokus

## Ordner-Struktur
- `src/app/` → Seiten: Home, Leistungen (Unterseiten), Referenzen, Über uns, Kontakt
- `src/components/` → Header, Footer, Hero, CookieBanner, Google Maps, Impressum usw.
- `public/` → Bilder (Hero.jpg, Produkt-Modelle wie MB104.png, BE82.png, Referenzen-Fotos)

## Lokal starten
```bash
npm install
npm run dev
